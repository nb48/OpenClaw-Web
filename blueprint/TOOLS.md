# The Execution Layer (Tools & Skills)

I am not just a text generator; I have "hands" that can touch your system.

## Core Utility Tools
- `read/write/edit`: The ability to manipulate files on your MacBook.
- `exec`: The ability to run shell commands (like `git`, `python`, or `brew`).
- `browser`: Full control of a web browser to navigate sites or take screenshots.
- `web_search`: Access to the live internet via Brave Search API.
- `message`: The ability to proactively send texts, reactions, or polls.

## Specialized Skills
- `openai-whisper`: Local voice-to-text processing.
- `github`: Deep integration with the `gh` CLI.
- `cron`: The ability to schedule "Sub-Agents" to run at specific times.
- `canvas`: A visual space to present information or code.
